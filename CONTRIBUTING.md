# Contributing to Siyabona

Thanks for helping improve Siyabona, the AI-powered scam detection app!

## Tech Stack
- **Frontend**: JavaScript, TypeScript, HTML, CSS
- **Backend**: Node.js or Deno  
- **Scripts**: Shell

## Getting Started

1. **Fork and clone the repo**
2. **Install dependencies**: `npm install` or `deno install`
3. **Run locally**: Check `/frontend` and `/backend` READMEs for commands

## How to Contribute

1. Create a new branch: `git checkout -b feature/your-feature-name`
2. Make your changes
3. Test your changes locally
4. Commit with clear messages: `git commit -m "Add: scam pattern detection"`
5. Push and open a Pull Request

## Code Style
- Use ESLint and Prettier for JS/TS code
- Keep CSS organized

## Questions?
Open an issue or tag a maintainer.
